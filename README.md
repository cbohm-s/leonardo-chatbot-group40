 Leonardo Chatbot
A web-based chatbot recreating Leonardo Da Vinci using a scripted, persona-driven conversation engine.  
Built as part of a Software Engineering group project.

Project Overview:

This project implements a chatbot that responds in the character of Leonardo da Vinci.  
It uses a React frontend and an Express backend with conversation logic driven by structured JSON scricpts.
